the contract

Run a node. Earn credits. Spend them anywhere on the mesh.

CE is primitives, not a walled garden: identity, a libp2p mesh, a PoW chain + credit economy, sandboxed execution, content-addressed storage, and capability-based authorization. Everything else is an app on top.

Trustless by economics

A torrent-style share ratio rewards contribution; a settlement burn makes Sybil wash-trading cost real, unrecoverable capital. No admin to police it.

Capabilities, not accounts

Authorization is one primitive: signed, attenuating capability chains. Mint a scoped, expiring, revocable grant — no central ACL, no stored ip:port.

Your data, on your LAN

Mesh-first and air-gappable. Run distributed inference across a hospital's machines where PHI never leaves the building — something the cloud can't offer.

the toolkit

Typed, runtime-agnostic, on npm.

One client for the whole mesh — Node, Deno, Bun, browsers, edge Workers. Money is bigint base units (never floats); streams are typed async iterables.

place-a-job.ts
import { CeClient, Amount } from "@ce-net/sdk";

const ce = CeClient.local();          // http://127.0.0.1:8844
const status = await ce.getStatus();
console.log(`height ${status.height} · balance ${status.balance}`);

// find a GPU host and run a container on it, mesh-routed
for (const h of await ce.atlas()) {
  if (h.hasTag("gpu")) {
    const job = await ce.jobs.meshDeploy(h.nodeId, {
      image: "alpine:latest", cmd: ["echo", "hi"],
      cpuCores: 1, memMb: 128, durationSecs: 60,
      bid: Amount.fromCredits(10),
    });
    console.log(`deployed ${job} on ${h.nodeId}`);
    break;
  }
}
