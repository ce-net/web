# CE Chat

A realtime group chat showcasing the CE app platform: the `lobby` mesh room (`ce.room('lobby')`) carries live messages and presence (who's online) over gossipsub, while the CE database (`ce.db.set` / `ce.db.list`, a mesh-replicated map) keeps a durable message history that reloads on refresh. Each browser gets a stable public id and a chooseable name; messages show timestamps and sender. Polished, mobile-first dark UI in the CE aesthetic. The bundled `src/ce.ts` is the mesh-native `@ce/client` and talks only to your local node (same-origin), so a strict CSP holds. `npm i && npm run build` produces a static `dist/`; run it locally with `ce-app serve` or publish over the mesh with `ce-app expose` so it serves at `<name>.user.ce-net.com` via mesh ingress.
