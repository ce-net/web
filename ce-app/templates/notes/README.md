# CE Notes

A collaborative notes board / guestbook that proves CE persistence + multiplayer in one screen: every note is durably stored in the CE database (`ce.db.set` / `ce.db.list` / `ce.db.del`, a mesh-replicated map namespaced per app) so it survives refreshes and restarts, while a mesh room (`ce.room('notes')`, gossipsub) instantly fans new and deleted notes out to every other open tab. Sign your notes with a name, post with Cmd/Ctrl+Enter, delete your own. The bundled `src/ce.ts` is the mesh-native `@ce/client` and talks only to your local node (same-origin), so a strict CSP holds. `npm i && npm run build` emits a static `dist/`; run it locally with `ce-app serve` or publish over the mesh with `ce-app expose`.
