# CE Board

A multiplayer live-cursors whiteboard built purely on CE mesh rooms — no database, intentionally ephemeral. Every pointer move and pen stroke is broadcast over `ce.room('board')` (gossipsub) to everyone else viewing the same app, so you see named, colored cursors gliding around and ink appearing in real time; "Clear" wipes the canvas for the whole room. Coordinates are normalized so it lines up across different screen sizes, and it's touch-friendly. The bundled `src/ce.ts` is the mesh-native `@ce/client` and talks only to your local node (same-origin), so a strict CSP holds. `npm i && npm run build` produces a static `dist/`; run it locally with `ce-app serve` or publish over the mesh with `ce-app expose`.
