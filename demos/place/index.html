Each tile is its own authoritative shard. The host enforces a per-identity cooldown and a strict per-pixel write order, then broadcasts the tile over gossipsub — and periodically seals the canvas + per-painter credits against the PoW chain so the result is tamper-proof and verifiable by a stranger.
